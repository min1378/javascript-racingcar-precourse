# 🏎️ 자동차 경주 게임

## 🎯 개발해야 할 기능

- 자동차 이름 입력 받기
  - [ ] 쉼표(,) 를 기준으로 구분하기
  - [ ] 이름은 5자 이하만 입력(예외 처리)

- 이동 횟수 입력 받기
  - [ ] 숫자를 입력받기 (1 ~ 50) 

- 자동차 움직이기
  - [ ] 움직일지 말지 판단하는 random 함수 구현 (4이상 이동, 3이하 정지)
- 실행 결과 표현하기
  - [ ] 횟수만큼 실행결과 반복해서 보여주기
  - [ ] 자동차 이동한 만큼 하이픈(-)으로 표현하기
  - [ ] 횟수만큼 반복 후, 최종 우승자 출력하기(동점자는 쉼표(,)로 구분하여 표현)
